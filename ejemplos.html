Bloque 2 - Retardo a la desconexión

Este bloque funciona de tal manera que la entrada Trg define si el bloque "Retardo a la conexión" se activa o no, siendo que si I1 = 1, este se activa, pero si I = 0, este se desactivara tras pasar un tiempo dado en los parametros del bloque, puede ser de 5 segundos, asi como 5 minutos, o incluso hasta horas. En dado caso que este se encuentre activado, pero I1 = 0, al pulsar "Reset" lo que haremos es que este se apague, cambiando su estado a 0 sin importar si el tiempo colocado haya pasado o no, dejandolo desactivado. Si llegase a ocurrir que I1 =1 e I2= 1, este se mantendra desactivado hasta que I2 = 0 (La entrada al reset se desactive). Cuando I1 = 1 y el temporizador ya estaba corriendo, este se reiniciara, volviendo a empezar desde 0 cuando I1 vuelva a 0.
